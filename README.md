## üìò Overview
The **Blood Bank Management System** is a menu-driven mini project developed in **C language** to manage and organize donor records efficiently.  
It allows users to add, view, search, update, and delete donor information ‚Äî all stored permanently in a text file (`bloodbank.txt`).

This project demonstrates how **file handling** and **structured programming** can be used to build a simple yet practical system for managing critical data.

---

## ‚öôÔ∏è Features
- Add new blood donor details  
- Display all existing donors in a structured format  
- Search donors by **name** or **blood group** (case-insensitive)  
- Update donor details like name, age, gender, contact, or city  
- Delete donor records from the file  
- Data stored permanently in `bloodbank.txt` for future reference

- **Programming Language:** C

  - File Handling (`fopen`, `fclose`, `fprintf`, `fscanf`)

  - Menu-Driven Programming
